# Philodex

> *"Attrapez-les tous — par la raison."*

Un Pokédex de la philosophie. Les philosophes sont des **Philomon** à collecter, chacun avec ses types, ses statistiques, ses concepts-attaques et sa filiation.

## Fonctionnalités

- **8 Philomon** : Descartes, Kant, Hegel, Hume, Sartre, Nietzsche, Mill, Épictète
- **Sprites pixelisés** générés depuis portraits historiques (Gen I · Game Boy / Gen II · GBC / Gen III · GBA)
- **Système de types** par champ philosophique (Épistémologie, Métaphysique, Éthique, Politique, Existentiel)
- **Matchups** : forces et faiblesses entre courants d'un même champ — incommensurabilité entre champs différents
- **Concepts-attaques** avec score de force et efficacité typologique
- **Filiation** : maîtres, disciples, oppositions
