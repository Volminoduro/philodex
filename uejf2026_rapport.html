Note méthodologique : Les scores sont exprimés en % des répondants ayant déclaré être « D'accord » avec l'affirmation. Pour l'indicateur d'antisémitisme, la base est restreinte aux répondants ayant répondu à toutes les affirmations (hors NSP), soit n=1 331. Les colonnes « le plus élevé / le plus bas » sont calculées parmi les partis détaillés dans le rapport : LFI, Parti Socialiste, Les Écologistes (EÉLV), Renaissance, Les Républicains, Rassemblement National, et Sans sympathie partisane. Les tirets (—) indiquent une valeur nulle ou non disponible dans le rapport.
